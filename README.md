# Currency Converter - Конвертер валют
**Автор:** Жук Ярослав
--
## О проекте 
десктопное приложение для конвертации валют.
Получает актуальные курсы через **CurrencyAPI.com**, сохраняет историю операций в JSON.
--
# Как получить API-ключ
1. Перейдите на сайт: **https;//currencyapi.com/**
2. Нажмите **Get Started**
3. Зарегистрируйтесь через Google, GitHub или Email
4. Подтвердите Email
5. После входа в личный кабинет скопируйте ключ из раздела **Dashboard**
6. Вставьте ключ в переменную 'API_KEY'
